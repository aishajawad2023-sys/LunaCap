# 🌙 LunarCap AI

LunarCap AI is a Streamlit-based engineering application that predicts the stability of capillary-driven fluid transport using the Young–Laplace equation.

## Features

* Physics-based capillary pressure calculation
* Stability prediction (Stable, Marginal, Dry-Out Risk)
* Interactive engineering dashboard
* Pressure vs. pore radius visualization
* Ready for future AI-powered engineering explanations

## Technologies Used

* Python
* Streamlit
* NumPy
* Pandas
* OpenAI API (optional)

## Future Work

* Integrate machine learning models trained on simulated low-gravity transport data.
* Enhance the AI explanation layer with engineering recommendations.
